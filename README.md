# 🚀 STAKE SENSE
> AI-Powered eConsultation Platform for Intelligent Stakeholder Feedback Analysis

STACK SENSE is an AI-powered eConsultation platform developed for **Smart India Hackathon (SIH) 2025**. The platform helps government organizations and policymakers efficiently analyze stakeholder feedback on draft legislations and policy proposals using Artificial Intelligence.

By leveraging AI techniques such as **Sentiment Analysis**, **Summary Generation**, and **Word Cloud Visualization**, STACK SENSE significantly reduces the manual effort required to review thousands of public comments while ensuring that every suggestion is systematically analyzed.

## 📖 Overview

The eConsultation Analysis System helps government organizations efficiently manage public consultations on draft legislations. Instead of manually reviewing thousands of stakeholder comments, the platform uses AI to automatically classify sentiments, summarize feedback, and generate keyword-based visualizations, enabling faster and more informed decision-making.

## 🎯 Problem Statement

Large numbers of stakeholder comments submitted during public consultations make manual review difficult and time-consuming.

This project uses Artificial Intelligence to automatically:

- Analyze stakeholder sentiments
- Generate concise summaries
- Extract important keywords
- Visualize discussions using word clouds

This significantly reduces manual effort while ensuring that every stakeholder suggestion is properly considered.
